How to Embed ConvoForm in Your Website

Follow these simple steps to embed the ConvoForm in your website:

  1. Include the ConvoForm CSS and JavaScript files from the dist folder
  2. Create a container element where you want the form to appear
  3. Initialize the form with your configuration

Implementation Code

Here's the code used to embed the form above:

<!-- Include ConvoForm CSS and JS -->
<link rel="stylesheet" href="./dist/assets/index-p7f-vyxE.css">
<script src="./dist/assets/index-xWriA2Qd.js" defer></script>

<!-- Create container for the form -->
<div id="convoform-container"></div>

<!-- Initialize ConvoForm -->
<script>
  document.addEventListener('DOMContentLoaded', function() {
    // Wait for ConvoForm to be loaded
    setTimeout(function() {
      if (window.ConvoForm) {
        // Initialize with configuration
        window.ConvoForm.init({
          targetElement: '#convoform-container',
          configUrl: './public/demo-config.json',
          onComplete: function(data) {
            console.log('Form completed with data:', data);
            // You can send this data to your server or process it as needed
          }
        });
      }
    }, 500);
  });
</script>
